使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可
這樣用最好用
身體保養
  • # 肌肉痠痛：10ml / 純正薰衣草2 + 甜馬鬱蘭2 + 檸檬2
  • # 提升睡眠品質：10ml / 純正薰衣草2 + 苦橙葉2 + 佛手柑2
  • # 舒緩敏感肌膚：10ml / 純正薰衣草2 + 乳香2 + 羅馬洋甘菊2
  • # 腸胃不適：10ml / 純正薰衣草2 + 甜羅勒1 + 甜橙3
情緒調理
  • # 調整心情，提升幸福感：純正薰衣草+甜橙+天竺葵+佛手柑
  • # 空間除菌除臭：純正薰衣草+檸檬+綠花白千層
  • # 保養呼吸道：純正薰衣草+澳洲尤加利+月桂
肌膚護理
  • # 臉部痘痘肌：10g 乳液精油配方 / 純正薰衣草1 + 乳香1 + 大西洋雪松1
  • # 濕疹：10ml / 純正薰衣草2 + 廣藿香1 + 德國洋甘菊3

注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢
常見問題
Q1：可以天天使用精油嗎？
A：雖然精油可以天天使用，但不宜長期使用同一種精油，因為可能會累積過多在體內，會有不良影響，且使用太頻繁而使身體對這個精油產生鈍化現象，而失去療效。
