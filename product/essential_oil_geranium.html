－天竺葵精油

開著粉色小花的天竺葵氣味和玫瑰相像，價格卻比玫瑰精油平易近人許多，所以天竺葵精油又有「窮人的玫瑰」之稱。
雖然天竺葵精油的「陰柔特質」沒有玫瑰那麼強烈，但也是很適合女性的精油，對於婦科、情緒、保養等方面都有很好的作用。濃郁豐富的香氣容易被女性喜愛，也常添在保養品之中。

為什麼你需要天竺葵精油？
  • # 平衡皮脂腺的分泌，還具有殺菌和收斂的效果，幾乎適合所有膚質使用。
  • # 刺激腎上腺皮質的分泌，進而調控荷爾蒙的分泌，所以天竺葵精油對荷爾蒙變化造成的婦科問題有很好的效果。
哪些人最需要天竺葵精油？
  • # 肌膚有初老現象的人：擁有甜美花香的天竺葵對毛孔阻塞、鬆垮的肌膚很有幫助，也能使失去光澤的皮膚回復紅潤光彩，讓人看起來容光煥發。
  • # 有經前症候群的人：女兒大姨媽來前都渾身不對勁，媽媽更年期的熱潮紅、盜汗、情緒低落，母女一起使用天竺葵精油，全家氣氛都變得和樂融融。
使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可
這樣用最好用
身體保養
  • # 經前症候群、產後乳房腫脹不適：20ml 基底油 + 天竺葵精油5 + 橙花精油2 + 薰衣草精油3
  • # 月經來臨前幾天的脹痛不適：10ml甜杏仁油 + 天竺葵4滴 + 薑2滴
    註：擦在下腹部按摩，可以舒緩脹痛感
情緒調理
  • # 更年期調節婦科、情緒穩定：快樂鼠尾草、天竺葵、苦橙葉 （擴香幫助入眠）
肌膚護理
  • # Ｔ字部位易出油，毛孔總是say hi：10ml杏桃核仁油 + 天竺葵1 + 橙花1 （此為1%濃度可用在臉部肌膚）

注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢
