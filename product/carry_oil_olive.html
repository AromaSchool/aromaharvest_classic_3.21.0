商品介紹
乾皮必備基底油
－橄欖油

冷壓初榨的橄欖油品質最好，含亞麻油酸、橄欖多酚與植物固醇等養分，並帶有特殊的果實香氣。 除了是天然的防曬油，滋潤的質感也非常適合乾性皮膚使用，能加強肌膚保濕，達到穩定緩和的效果。

點我進一步認識橄欖油 >
為什麼你需要橄欖油？
  • # 緩解皮膚乾燥缺水、痕癢刺痛或發炎的情況。
  • # 天然防曬，最高能阻隔20%的太陽光線。
  • # 作為按摩油，潤澤順滑。
哪些人最需要橄欖油？
  • # 皮膚乾燥的人：滋潤度高的橄欖油，能幫助潤澤角質、軟化肌膚，達到鎖水保濕的效果。
  • # 皮膚有瑕疵的人：持續塗抹橄欖油能淡疤抗皺，幫助修復皮膚創傷。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以1:9的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 妊娠紋、防曬、抗老化美容護膚的極佳選擇。
  • # 可100%使用，或與其他基底油調和。

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
