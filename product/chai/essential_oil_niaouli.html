－綠花白千層精油

綠花白千層白色絨毛狀的花序，讓人聯想到可愛的小瓶刷，精油萃取自葉片及幼枝，其氣味具有穿透性，帶著點薄荷及樟腦味。 在法國的醫療系統中常被用來消毒殺菌，它也是許多藥劑中常用的成份。
在馬來西亞及爪哇，綠花白千層為十分普遍的藥用植物，南洋地區的居民則以其果實增進食慾。特別注意：孕期不可以使用，小孩、嬰兒也不宜。

為什麼你需要綠花白千層精油？
  • # 可以協助掃除、收乾上下呼吸道的黏液，舒緩胸腔的閉塞感。
  • # 處理呼吸系統感染，舒緩感冒咳嗽、喉嚨痛、肺炎和鼻竇炎。
  • # 可殺菌、抗感染、抗病毒。
  • # 有助於改善風濕關節炎和腸胃炎、脹氣及潰瘍。
哪些人最需要綠花白千層精油？
  • # 面皰、異位性皮膚炎困擾的人：綠花白千層能平衡油脂分泌，改善油性肌膚及頭皮的油脂分泌量。
  • # 想要環境消毒的人：綠花白千層與迷迭香或檸檬等抗菌精油製成酒精噴霧，外出使用消毒殺菌效果加乘。
  • # 泌尿道感染的人：適用於所有泌尿道的感染，如膀胱炎、念珠菌感染、分泌物等情況。
使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法：基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可

  • # 提神醒腦，強化呼吸防護力配方：10ml基底油 + 澳洲尤加利3 + 綠花白千層2 + 茶樹1

注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢
常見問題
Q1：可以天天使用精油嗎？
A：精油可以天天使用，但每款精油成分不同，對人體的影響也有所差異。本網站提供的用法皆溫和安全，且經專業芳療師確認，原則上固定配方使用三周要休息一周，身體才不會逐漸習慣，並能讓精油發揮最大效果。
