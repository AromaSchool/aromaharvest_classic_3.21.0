為肌膚而生的基底油
－荷荷芭油

與肌膚皮脂構造相似的荷荷芭油，非常親膚好吸收，可以說是任何膚質都適用的基底油。
如果你對氣味敏感又想要油脂成分裡的豐富營養，無異味的荷荷芭油絕對是最佳選擇！

點我進一步認識荷荷芭油 >
為什麼你需要荷荷芭油？
  • # 滋潤肌膚、預防乾燥，並使肌膚維持柔嫩
  • # 天然的抗氧化劑，穩定性極佳，不易變質和產生油耗味
  • # 防曬係數SPF4，可做為護唇和baby的天然防曬油
哪些人最需要荷荷芭油？
  • # 選擇障礙不知要選哪基底油的人：如果你想買一瓶穩定性極高，不易產生油耗的基底油，荷荷芭油就是直接勾選的最好的選擇。
  • # 想要一瓶油萬用的人：荷荷芭油可深入角質層，親膚的保養油，想要直接拿來卸妝，或者調成按摩油作臉部按都很好用。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 改善菜瓜布肌膚：10ml荷荷芭油 + 玫瑰原精6
  • # 滋潤稻草髮絲：直接抹在微濕的髮尾可以讓頭髮恢復柔順。
肌膚護理
  • # 臉部抗老美容油：10ml荷荷芭油 + 義大利永久花1 + 玫瑰1 + 花梨木1
  • # 肌肉僵硬按摩油：10ml荷荷芭油 + 薑1 + 檜木3 + 桉油醇迷迭香2

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
