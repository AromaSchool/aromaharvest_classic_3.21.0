神聖之花的基底油
－金盞花油

印度人尊奉金盞花為神聖之花，以乾燥金盞花浸泡而成的金盞花油呈美麗的黃色，不管是色澤或功效都非常溫和舒服。
含豐富的胡蘿蔔素、葉黃素和植物固醇，可處理各種皮膚問題，特別是皮膚乾裂、疤痕、創傷、潰爛消腫、去瘀、靜脈曲脹以及擦傷等問題。

為什麼你需要金盞花油？
  • # 有不容易處理好的傷口時
  • # 有任何皮膚症狀的問題
  • # 淡化疤痕
哪些人最需要金盞花油？
  • # 討厭化學介入保養的人：有時候太多的化學介入反而是導致肌膚敏感的原因，具有修復效果的金盞花油是敏感肌膚最天然的草本選擇。
  • # 經常有皮膚常見問題：皮膚老是有些小毛病，金盞花油擅長處理各種皮膚問題，與其他基底油調和塗抺在脆弱敏感處，讓肌膚慢慢自然恢復。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 濕疹、乾癬：2ml金盞花油 + 8ml甜杏仁油 + 苦橙葉1 + 玫瑰草2 + 德國洋甘菊3
  • # 尿布疹：滴一滴在手心後與乳液混和然後塗抹在屁股即可
  • # 淡化疤痕：金盞花油2ml + 澳洲胡桃油8ml + 純正薰衣草3
肌膚護理
  • # 提升光澤和拉提美容油：金盞花油20% + 甜杏仁油80%調和即可
注意事項
  1. 請放置冰箱冷藏
  2. 保存期限依瓶身標示，請於期限內使用完畢
常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
