保護脆弱敏感肌的基底油
－酪梨油

酪梨油採自酪梨果實，本身質地稍厚滋潤度很高，也有很高的營養價值，適合乾性、敏感性及老化肌膚，能使肌膚重回柔軟及潤澤的狀態；
它的滲透力僅次於荷荷芭油，因此有不錯的深層清潔效果。

為什麼你需要酪梨油？
  • # 具有促進細胞再生功能，可改善乾性、脆弱及敏感性肌膚。
  • # 含豐富脂溶性維生素，並有豐富的卵磷脂和少許的胡蘿蔔素，具有很好的保濕滋潤效果。延展性佳，油質厚重很適合當按摩油，尤其適合冬天當護膚油使用。
  • # 滋潤及修復效果好，有助於鎖住水分安撫受損肌膚。
哪些人最需要酪梨油？
  • # 因為有小細紋而虛長年齡的人：天生乾性肌惱人的小細紋讓人很顯老，酪梨油具有修復及滋潤保水作用，好好按摩減齡不是夢。
  • # 頭髮暗淡無光的人：頭髮乾的像稻草，連鳥兒都想來築巢，酪梨油能有效修復及保濕，恢復動人髮絲。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以1:9的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 保濕滋潤配方: 10ml酪梨油10% + 杏桃核仁油90% + 純正薰衣草1 + 茉莉原精1 +花梨木1
    ( 註：因油質厚重，適合與其他基底油調和，最高佔基底油的25%即可。 )

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
