－葡萄柚精油

這麼多的精油中，柑橘類精油是最為大眾所接受的，其中氣味清甜豐潤的葡萄柚精油也是最被人喜愛的精油之一。它不僅味道討喜，也具備許多的功效。 它能夠促進淋巴循環、激勵肝臟分泌膽汁、調節消化系統，因此常被用來消除水腫、改善橘皮組織，是很多人「減肥」必備的精油聖品。
另外，葡萄柚精油中所含的檸檬稀能夠激勵神經，促進多巴胺分泌，安撫沮喪、低落的情緒。

為什麼你需要葡萄柚精油？
  • # 促進淋巴流動的效果極佳，幫助排除多餘水分，使身體輕盈。
  • # 改善食慾不振、幫助消化。
  • # 提振情緒，讓心情好起來。
哪些人最需要葡萄柚精油？
  • # 容易水腫的人：葡萄柚精油具有利尿、促循環的特性，能針對水腫症狀發揮效果。
  • # 食慾不振的人：氣味香甜、幫助開胃，也能促進消化。
  • # 需要調整時差的人：具有提振、激勵神經的效果，促進腦內啡分泌，舒緩倒時差及水土不服所帶來的頭痛、疲乏。
  • # 壓力過大的人：平衡身心，讓心態變得放鬆，解除自己施加給自己的壓力。
使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法：基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可
這樣用最好用
身體保養
  • # 水腫byebye配方：荷荷芭油10ml + 葡萄柚4 + 杜松2，洗完澡後塗抹在下半身。
情緒調理
  • # 心情不好沒食慾：芝麻油10ml + 葡萄柚2 + 維吉尼亞雪松1 + 絲柏2 + 檀香1 ，加強在腰腹及下半身的按摩。

注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢

Q1：可以天天使用精油嗎？
