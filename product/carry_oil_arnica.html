崇尚自然的歐洲跌打損傷油
－山金車花油

山金車花的草藥活性通過浸泡植物油萃取而出，有舒緩跌打損傷的效果，是普遍的家庭常備藥。
由於山金車花含毒，不建議直接使用於臉部或破損傷口。

點我進一步認識山金車花油 >
為什麼你需要山金車花油？
  • # 外用於受傷部位，如：扭傷、骨折、肌肉關節疼痛等，能抗炎止痛、消除腫脹。
哪些人最需要山金車花油？
  • # 腰酸背痛的人：含百里酚、內酯、山金車素、類黃酮等。適用於跌打損傷、活血化瘀、消炎、消腫止痛、減緩扭傷肌肉痛、韌帶拉傷、瘀青、疼痛、關節炎、運動傷害及車禍開刀後的長期復原。用於按摩能舒緩疼痛，加速痊癒。(不可使用在有傷口的部位)
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以1:9的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 單擦適合運動後按摩放鬆肌肉。
  • # 跌打損傷精油配方: 10ml山金車花油(或40%山金車花油 + 60%荷荷芭油) + 西洋蓍草2 + 甜馬鬱蘭2 + 台灣紅檜2
    ( 註1：荷荷芭油可以延長山金車花油的穩定性， 使油較不易氧化產生油耗味。如以價格考量也可以混和葵花籽油、橄欖油、杏桃核仁、澳洲胡桃油、甜杏仁油等。 )
    ( 註2：因價高通常與其他基底油合併使用。)

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
