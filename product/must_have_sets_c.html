－典藏10支精油組

典藏10支精油組是每個芳療新手必備的基本入門精油組合，精油品項包含：檸檬、甜橙、薄荷、桉油醇迷迭香、純正薰衣草、澳洲尤加利、茶樹、花梨木、綠花白千層、甜馬鬱蘭。
另外還有葵花油100ml一瓶、12格木盒、五支滾珠瓶。上列精油概括了大部分身心症狀上的處理，在運用上可以單支分別使用，也可以隨意組合、搭配。

為什麼你需要典藏10支精油組？
  • # 甜橙與檸檬屬於柑橘類精油，柑橘類精油氣味討喜，有使人放鬆的屬性，可以舒緩緊張的情緒與壓力，也常用來調理循環系統及消化系統。
  • # 純正薰衣草一向有「媽媽油」的稱號，具有鎮靜、消炎、止痛、助眠的效果，功效全面又溫和。
  • # 桉油醇迷迭香、澳洲尤加利、茶樹、綠花白千層皆以抗菌、抗病毒及強化免疫的功能為名，可以用來預防或處理感冒、過敏的問題。
  • # 花梨木為木質類精油，具有鎮靜神經、安撫情緒的作用，也常被作用在呼吸系統上，能夠緩解呼吸道不適。
  • # 葵花油對皮膚保養效果非常好，適合用來稀釋精油，也可以單獨直接作為按摩油使用。

使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法：基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可
這樣用最好用
身體保養
  • # 早上集中注意力：迷迭香2 + 檸檬1
  • # 晚上放鬆身心：薰衣草2 + 柑橘1
情緒調理
  • # 經典好眠組合：快樂鼠尾草 + 佛手柑 + 苦橙葉 + 花梨木
居家清潔
  • # 維持空氣清新：打掃前在水桶內滴入柑橘類的檸檬或甜橙 + 茶樹

注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢
常見問題
Q1：可以天天使用精油嗎？
A：精油可以天天使用，但每款精油成分不同，對人體的影響也有所差異。本網站提供的用法皆溫和安全，且經專業芳療師確認，原則上固定配方使用三周要休息一周，身體才不會逐漸習慣，並能讓精油發揮最大效果。
