嬰幼兒最好的按摩油
－甜杏仁油

甜杏仁油以冷壓榨取甜杏仁果實而成，營養價值高，親膚性好、容易吸收，是一種延展性好且溫和不油膩的基礎油，與任何植物油皆有調和互補的特性、因此也是芳療中經常搭配使用的油。
但是因含比例較高的亞硫酸，所以要小心氧化問題。

為什麼你需要甜杏仁油？
  • # 親膚性良好，長期使用可以有效地消除妊娠紋，對面皰也有調理作用。
  • # 以甜杏仁油按摩可以加強細胞帶氧功能，鎮痛及減輕因運動過度疲勞與碳酸累積引起之肌肉疼痛。
哪些人最需要甜杏仁油？
  • # 家裡有小北鼻的人：溫和低過敏性的特質是給北鼻最好的按摩油，也可舒緩尿布疹，讓媽媽溫柔的膚觸給予寶寶親密的安全感。
  • # 容易有黑頭粉刺的人：甜杏仁油容易吸收因此能深入皮膚，能軟化累積在毛孔中的污垢，是一款能防止惱人黑頭粉刺產生的基底油。
  • # 眼圈黑的像熊貓的人：提升細胞帶氧的功能，除了消除肌肉疲勞之外，睡覺前用少量使用於眼圍再好好搭配作息，就能早日和黑眼圈和眼袋說BYE BYE。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以1:9的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 可100%使用，甜杏仁油有著延展性良好的特性，能在按摩時提供較長時間的潤滑。
肌膚護理
  • # 適用混合型肌膚
  • # 可做卸妝油使用，全臉抹勻甜杏仁油，充分按摩溶解彩妝後以衛生紙吸附過多油脂，再以清水將殘妝沖洗乾淨。

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
