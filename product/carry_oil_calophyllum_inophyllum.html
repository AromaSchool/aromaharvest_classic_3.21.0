具療癒保護的基底油
－瓊崖海棠油

瓊崖海棠油是從果仁冷壓榨取而來的，但它整棵樹木都相當有價值，像是樹皮分泌出的樹脂被稱為「卡拉巴香脂」，可用來止血並作為局部麻醉。
瓊崖海棠油呈現油墨般暗綠色，有強烈藥草氣味，具有止痛及促進疤痕修復的消炎效果。

為什麼你需要瓊崖海棠油？
  • # 具有消炎、抗菌的特性，臨床上用來改善靜脈曲張、修復疤痕、處理感染肌膚。
  • # 對於處理神經痛、關節及肌肉疼痛有不錯的效果。
  • # 適用任何皮膚問題，包含靜脈曲張、促循環、傷口復原、疤痕、骨骼肌肉、鎮痛消炎、抗菌。
哪些人最需要瓊崖海棠油？
  • # 想要淡化疤痕的人：皮膚和肌肉經歷一些外來傷害都需要一些時間好好修復，讓瓊崖海棠油幫助促進傷口癒合，儘早恢復到最好的狀態。
  • # 經常久坐久站的人：久站是靜脈曲張最大的原因，運用瓊崖海棠油來穩定靜脈、活絡血流、使靜脈中的血液暢通，或者難以啟齒痔瘡等問題也有不錯的療效。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以1:9的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
這樣用最好用
身體保養
  • # 帶狀皰疹: 10ml瓊崖海棠油20% +荷荷芭油80% +羅文莎葉3 +佛手柑 3
    ( 註：適合與其他基底油調和使用，護膚10%以下，其他用途20%以下。 )
注意事項
  1. 請放置冰箱冷藏
  2. 保存期限依瓶身標示，請於期限內使用完畢
  3. 只供外用，不可內服
常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
