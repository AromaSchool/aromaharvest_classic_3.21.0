讓肌膚重返活力的修復油
－杏桃核仁油

帶著淡淡杏仁香氣的杏桃核仁油，取自杏桃的核(籽)，杏桃核仁油成份較多單一不飽和脂肪酸，質地較清爽也較容易被肌膚吸收，適合使用臉部按摩或是臉部產品中。
它也能和其他質地較滋潤的油調配，使質感感覺更為清新。

為什麼你需要杏桃核仁油？
  • # 杏核油含有豐富的維生素Ａ和Ｂ1、Ｂ2、Ｂ6、礦物質、不飽和脂肪酸及亞麻仁油酸，因此具有良好的親膚潤膚特質，對於肌膚的活化保水滋潤對都有不錯的效果。
  • # 杏核油質地細緻，相當適合作臉部按摩，能增加皮膚光澤，恢復彈性。
  • # 杏桃核仁油常用在改善動脈硬化和對抗關節風濕症狀。
哪些人最需要杏桃核仁油？
  • # 想告別黃臉婆的人：肌膚極敏感，常花大錢買保養品卻老是敏感發炎、杏桃核仁油的成分溫和用來按摩臉部能改善膚色蠟黃提升明亮氣色。
使用方式
單用
  • # 直接使用基底油塗抹或按摩在需要的部位。
調油
  • # 與精油按照濃度建議調和使用。
乳液
  • # 基底油與蘆薈膠以1:9的比例調成乳液。作法:基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可。

身體保養
  • # 可100%使用，比甜杏仁油更保濕，更適合用在皮膚按摩。
肌膚護理
  • # 適用混合型肌膚

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
