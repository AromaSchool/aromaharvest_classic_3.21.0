身體循環不佳的調理劑
－杜松精油

杜松是琴酒的主要成分之一，不同於萃取針葉/木頭的便宜等級，沐禾的杜松精油萃取自深藍色的成熟漿果。清爽的木質香氣，帶著一些辛辣和甜美，令人感覺溫暖鎮靜。
杜松精油是生殖泌尿道的抗菌劑，能幫助排除身體堆積的毒素，改善沉重疲憊的身體。

為什麼你需要杜松精油？
  • # 清血排毒，能維護肝腸健康、清除尿酸、舒緩風濕和關節炎。
  • # 改善體內水分囤積，幫助排尿、催汗、舒緩水腫。
  • # 調理油性皮膚及頭皮。
哪些人最需要杜松精油？
  • # 身體循環差的人：舒緩身體毒素累積而導致的腰酸背痛和四肢浮腫，同時激勵和強化神經，讓身體變得更輕盈。
  • # 肥胖族群：改善循環系統的同時，能幫助調節胃口、強化四肢。
使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法：基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可
這樣用最好用
身體保養
  • # 呼吸防護：大西洋雪松2 + 檜木2 + 杜松2 (擴香)
情緒調理
  • # 強化精神：基底油 4.5ml + 綠花白千層3 + 杜松1 + 澳洲尤加利1

注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢
常見問題
Q1：可以天天使用精油嗎？
A：精油可以天天使用，但每款精油成分不同，對人體的影響也有所差異。本網站提供的用法皆溫和安全，且經專業芳療師確認，原則上固定配方使用三周要休息一周，身體才不會逐漸習慣，並能讓精油發揮最大效果。
